# Excel Analysis Plan

Excel is useful for quick checks before building the full dashboard or presenting findings.

## Practical Uses

- Validate row counts between raw and cleaned data.
- Check duplicate order IDs and confirm removal logic.
- Create quick pivots for city, cuisine, discount band, and order status.
- Spot-check gross profit calculations on a small sample.
- Review missing values and category cleanup before loading into Power BI.
- Share simple KPI checks with non-technical stakeholders.

## Pivot Checks

| Pivot | Purpose |
|---|---|
| Orders by city and month | Check growth pattern and city coverage |
| Gross profit by cuisine | Identify profitable categories |
| Delay rate by zone | Find operational pressure points |
| Repeat rate by discount band | Check whether discounts help retention |
| Average rating by delay flag | Validate service impact story |
